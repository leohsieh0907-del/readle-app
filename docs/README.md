## 一句話定位

> 用 AI 助教，把英文學習變成每天 10 分鐘的優雅習慣。

## 目標使用者

- 上班族通勤 10–20 分鐘自學英文
- 多益準備族（550 分想衝 750+）
- 商業英文需求（會議、Email、面試）
- 一般英文興趣愛好者

## 設計原則

1. **柔和優雅** — 不要 Duolingo 那種高飽和螢光色，要 Apple 的克制與留白
2. **資訊清晰** — 每頁只有一個明確 CTA，學習路徑零思考成本
3. **動畫有意義** — 動畫服務於資訊回饋（正確 / 錯誤 / 進步），不是炫技
4. **手機優先** — 通勤場景使用，所有功能在手機上要可單手操作
5. **AI 是助手不是主角** — AI 助教隨叫隨到，但不打擾學習節奏
