## 3.1 首頁 Dashboard（/）

### 3.1.1 區塊組成（由上而下）

| # | 區塊 | 內容 | 互動 |
|---|---|---|---|
| 1 | Hero 問候卡 | 「早安／午安／晚安，{暱稱}」+ AI 助教一句話建議 | 點擊進入 AI 助教 |

| 6 | 最近紀錄 | 最近 5 筆學習活動的小列表 | 點擊回到該活動 |

### 3.1.2 首次進站體驗

新使用者首次進站時，Hero 替換為**3 步驟引導卡**：
1. 選擇等級（初級 A1–A2 / 中級 B1–B2 / 高級 C1–C2）
2. 選擇興趣（多益 / 商業 / 日常 / 旅遊 / 科技 — 可複選）
3. 設定每日目標（10 / 20 / 30 / 60 分鐘）

完成後寫入 LocalStorage 的 `user.preferences`。

### 3.2.2 功能細節

**中英雙字幕**
- 顯示模式：英上中下 / 中上英下 / 僅英 / 僅中 / 全關
- 字級可調（小 / 中 / 大）
- 字幕點擊任一單字 → 跳出 `WordPopup`（釋義、音標、發音、加入單字本）
- 字幕區同步高亮目前播放句

**播放速度**
- 0.5x · 0.75x · 1.0x · 1.25x · 1.5x · 1.75x · 2.0x
- 7 段，按 + / − 鍵切換

**AB Repeat**
- 點 A 設起點 → 點 B 設終點 → 循環播放此段
- 顯示 `🔁 0:14 ↔ 0:23 (×3)` 在控制列
- 可設定重複次數（3 / 5 / 10 / ∞）

**自動跟讀模式**
- 開啟後：影片每講完一句 → 自動暫停 → 提示「換你說」
- 錄音 → 顯示自己的波形 vs 原音波形
- 完成後自動播放下一句

**AI 發音評分**
- 錄音上傳 → AI 分析（音準、流暢度、節奏）
- 給 0–100 分 + 文字回饋（哪個音節需加強）
- Phase 1–2 使用 mock 隨機分數；Phase 3 接入

**控制列其他**
- ⌨ 快捷鍵：空白（播放）、← → （±5s）、↑ ↓（音量）、R（重播本句）

```
│  [Daily      80]               │
│  [Travel     22]               │
│  [Tech       20]               │
╰────────────────────────────────╯
```

### 3.3.2 單字卡模式（/learn/vocab/cards）

- 全屏卡片 + 翻面動畫（CSS 3D rotateY）
- 正面：英文單字 + 音標 + 發音鈕
- 背面：中文釋義 + 例句 + AI 例句生成鈕
- 左滑：標記「忘了」（明天再見）
- 右滑：標記「記得」（延後複習）
- 上滑：標記「已掌握」（移出複習池）
- 下滑：查看更多例句

### 3.3.3 SRS 記憶曲線複習（/learn/vocab/review）

採用 **SM-2 演算法**：

```
每個單字記錄：
- easeFactor       初始 2.5
- interval         下次複習天數
- repetitions      連續答對次數
- nextReviewDate   下次該複習的日期

複習時答題：
- 「忘了」     → repetitions = 0, interval = 1, ease -= 0.2
- 「模糊」     → interval *= 1.2, ease 不變
- 「記得」     → interval *= ease, repetitions++, ease += 0.1
- 「太簡單」   → interval *= ease * 1.3, ease += 0.15

每日複習：nextReviewDate <= today 的單字進池
```

### 3.3.4 AI 例句生成

每張單字卡背面有「再生一句」按鈕：
- 點擊 → AI 依使用者程度產生 1 句新例句
- 例句包含目標單字 + 1–2 個學過的單字（個人化）
- 例句可加入「我的例句本」

### 3.3.5 分類與標籤

預設 5 大類（TOEIC / Business / Daily / Travel / Tech），使用者可自訂標籤（如「面試」「電影台詞」）。

## 3.4 測驗中心（/quiz）

### 3.4.1 四種測驗類型

| 類型 | 題目形式 | 來源 |
|---|---|---|
| 聽力測驗 | 播放音檔 → 4 選 1 | AI 自動朗讀題目 |
| 填空題 | 句子挖空 → 從 4 個選項選 | AI 從你收藏的單字組句 |
| 單字測驗 | 中譯英 / 英譯中 / 看圖 / 同義詞 | 你收藏 + 已學過的單字 |
| AI 自動出題 | 綜合題型 | 根據近期錯題與弱點 |

### 3.4.2 答題介面

```
╭────────────────────────────────╮
│  Q3 / 10              ⏱ 0:24  │
│  ──────────────                │
│                                │
│  Choose the best meaning of    │
│  "elaborate"                   │
│                                │
│  ◯ A. 簡短的                   │
│  ◯ B. 詳細闡述                 │
│  ◯ C. 困難的                   │
│  ◯ D. 短暫的                   │
│                                │
│  [上一題]      [下一題 →]      │
╰────────────────────────────────╯
```

- 選錯 → 紅色閃光 + 抖動 + 顯示正確答案
- 選對 → 綠色閃光 + 縮放 + 自動跳下一題
- 答完所有題 → 結果頁

## 3.5 個人中心（/me）

### 3.5.1 頂部資訊

- 頭像（含等級邊框光暈）+ 暱稱 + 等級徽章
- 等級進度條（距下一級還差 X XP）
- 連續學習火焰（最長 vs 目前）

### 3.5.2 學習熱力圖

GitHub 風格 365 天熱力圖：
- 顏色越深表示當日學習時間越長
- 點擊任一日 → 看當日學習紀錄
- Hover → tooltip 顯示「2026-03-15 · 學習 24 分鐘」

## 3.6 設定（/settings）

| 區塊 | 設定項 |
|---|---|
| 帳號 | 暱稱、頭像（emoji 選擇器）、目標 CEFR 等級 |
| 學習偏好 | 每日目標分鐘、提醒時間、興趣分類 |
| 播放偏好 | 預設語速、預設字幕模式、字幕字級 |
| AI 偏好 | 模型（Gemini / Mock）、回覆語氣（嚴格 / 友善 / 幽默）、AI 程度（A1–C2） |
| 介面 | 亮 / 暗 / 跟隨系統、字型大小、減少動畫 |
| 資料 | 匯出我的資料（JSON）、清除全部資料 |
| 關於 | 版本、更新紀錄、隱私權 |

所有設定即時寫入 LocalStorage。
